# Business Context

## Problema
Recomendar restaurantes relevantes para o cliente sem comprometer viabilidade de entrega.

## Objetivo
Combinar preferencia do usuario com restricoes logisticas para aumentar conversao com boa experiencia.

## Perguntas de negocio
- Quais restaurantes devo mostrar para cada cliente agora?
- Quais opcoes maximizam chance de compra e entrega viavel?
- Qual a proxima categoria mais provavel para o cliente pedir?

## Regras de negocio iniciais
- Distancia maxima: 8 km
- ETA maxima: 45 min
- Taxa maxima de entrega: R$ 12,00
- Restaurante precisa estar aberto
