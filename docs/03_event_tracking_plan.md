# Event Tracking Plan

## Eventos essenciais
- restaurant_viewed
- menu_item_clicked
- cart_item_added
- checkout_started
- order_placed
- order_canceled

## Campos minimos por evento
- event_name
- user_id
- restaurant_id
- session_id
- timestamp
- user_location_lat
- user_location_lon

## Uso analitico
- Funil de conversao
- Recomendacao personalizada
- Diagnostico de abandono
- Analise de impacto de frete e ETA
